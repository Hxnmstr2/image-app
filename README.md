## Сборка и установка

1. Убедитесь, что у вас установлен Python 3.9+.
2. Создайте и активируйте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   source venv/bin/activate     # Linux/macOS
   venv\Scripts\activate        # Windows

3. Устанвовка зависимостей
    pip install -r requirements.txt
4. Запуск приложения:
  - Запуск из файла .py:
    Выполните из корня проекта "python app.py"
  - Запуск из .exe:
    Скачать app.exe, запустить скачаный файл :)


## Использование
- Кнопка "Загрузить изображение" — загружает изображение из файла.

- Кнопка "Сделать фото с камеры" — включает вебкамеру и делает мгновенный снимок, загружая его в приложение.

- Опция выбора канала (R, G, B) — показывать только выбранный цветовой канал.

- Кнопка "Обрезать" — обрезает изображение по указанным координатам.

- Кнопка "Добавить границу" — добавляет изображению рамку (грницу) указанной толщины.

- Кнопка "Нарисовать линию" — добавляет на изображении линию по указанным пользователем координатам.
